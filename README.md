# 🔍 commands 

   •  📸 `K.avatar` - Show your avatar and also with id or mention!
   
   •  🖼️ `K.icon` - Show the server icon!
   
   •  😂 `K.meme` - random reddit memes
   
   •  🏓 `K.ping` - Show KellyWorld ping!
   
   •  🌎 `K.setlanguage` - to change the server language!
